Claude Policy Runtime (CPR)

AI Treasury Analyst with Policy Governance
"The bottleneck to enterprise AI agents is not intelligence — it is delegated authority."
claude-policy-runtime — python3 run.py

How This Works

Data Layer READ — Claude queries treasury balances and market data (like Uber's Finch agent).

Action Layer WRITE — When Claude proposes a trade, the Policy Runtime evaluates it against human-readable rules and returns APPROVED / BLOCKED / ESCALATED.

Policies are markdown files. Non-engineers can edit rules in plain English. No code changes needed. This is "Decagon-style" agent operating principles for financial operations.
